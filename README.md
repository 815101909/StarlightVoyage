# 晓视界小程序

## 项目概述
晓视界是一个面向儿童的科普教育小程序，通过有趣的互动方式，帮助孩子们了解天文知识，培养科学兴趣。

## 技术栈
- 开发框架：微信小程序原生框架
- 开发语言：JavaScript
- 样式语言：WXSS
- 数据存储：微信小程序本地存储
- 开发工具：微信开发者工具

## 页面结构
从左到右依次为：

### 1. 星河 (pages/star/star)
- **功能描述**：展示天文知识、科普文章等内容
- **主要功能**：
  - 文章列表展示
  - 文章分类
  - 文章搜索
  - 文章收藏
- **技术要点**：
  - 列表懒加载
  - 搜索功能实现
  - 分类筛选

### 2. 观测 (pages/observe/observe)
- **功能描述**：天文观测相关功能
- **主要功能**：
  - 天文现象预报
  - 观测指南
  - 观测记录
  - 观测分享
- **技术要点**：
  - 天文数据展示
  - 观测记录管理
  - 分享功能

### 3. 星空 (pages/sky/sky)
- **功能描述**：星空探索和互动
- **主要功能**：
  - 今日星宿展示
  - 华夏星空：展示传统星图
  - 28宿卡片：展示二十八星宿的详细信息
  - 星宿功能：
    - 天官体系：展示星宿的官职体系
    - 天医体系：展示星宿的医疗体系
  - 星象表达：通过文学、占星、神话等方式表达星象
- **技术要点**：
  - 星空动画效果
  - 星宿数据管理
  - 轮播展示
  - A4打印区域布局

### 4. 探索 (pages/explore/explore)
- **功能描述**：天文探索和发现
- **主要功能**：
  - 天文发现
  - 探索任务
  - 成就系统
  - 探索记录
- **技术要点**：
  - 任务系统
  - 成就管理
  - 探索记录

### 5. 我的 (pages/profile/profile)
- **功能描述**：用户个人信息和功能入口
- **主要功能**：
  - 用户信息展示
  - 会员中心
  - 我的收藏
  - 观看历史
  - 功能入口：
    - 星空打卡：每日打卡功能
    - 星球许愿池：互动许愿功能
- **技术要点**：
  - 用户信息管理
  - 本地数据存储
  - 页面状态管理

#### 5.1 星空打卡 (pages/starcheckin/starcheckin)
- **功能描述**：每日星空打卡功能
- **主要功能**：
  - 每日打卡
  - 打卡日历
  - 打卡奖励
  - 打卡分享
- **技术要点**：
  - 日历组件
  - 打卡状态管理
  - 奖励机制

#### 5.2 星球许愿池 (pages/planet_wheel/planet_wheel)
- **功能描述**：互动许愿功能，让用户可以在星空中许下愿望
- **主要功能**：
  - 星空背景展示
  - 点击星空许愿
  - 愿望展示
  - 愿望管理
- **技术要点**：
  - 星空动画效果
  - 星星交互
  - 愿望数据存储
  - 日期记录

## 数据存储
- 使用微信小程序的本地存储 (wx.setStorageSync/wx.getStorageSync)
- 主要存储内容：
  - 用户信息
  - 打卡记录
  - 许愿记录
  - 观看历史
  - 收藏列表

## 开发规范
1. 文件命名：
   - 页面文件夹：小写字母，下划线分隔
   - 组件文件：小写字母，下划线分隔
   - 工具类文件：小写字母，下划线分隔

2. 代码规范：
   - 使用ES6+语法
   - 保持代码缩进统一
   - 添加必要的注释
   - 遵循小程序开发规范

3. 样式规范：
   - 使用rpx作为单位
   - 颜色使用变量管理
   - 保持样式模块化
   - 注意样式复用
